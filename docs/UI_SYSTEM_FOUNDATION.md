# 599549 UI 设计系统基础层

本次是在当前已开发状态上建立 UI 设计系统基础层。

不是回滚旧 HTML/CSS。
不是重做页面。
不是重构业务。

本次新增：

- public/assets/ui-system.css

命名空间：

- 前台：ui-front-*
- 后台：ui-admin-*
- 通用变量：--ui-*

本次原则：

- 业务不变
- 接口不变
- 数据库不变
- 权限不变
- 登录不变
- 开奖逻辑不变
- 澳门 / 香港同步考虑
- 共用后台继续共用
- 开奖数据继续独立
- 不删除旧 class/id/name/data-*
- 不改变 JS 选择器

后续规则：

- 后续只能逐页接入 UI class
- 不允许全站重构
- 不允许一次性替换全部 HTML/CSS
- 每次最多处理一个页面或一个模块
